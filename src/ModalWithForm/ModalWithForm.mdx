# ModalWithForm

> 提供一个带有 form 的 Modal，通过(按钮|文字|自定义元素)触发 Modal 的弹出

## Basic Usage

<Playground>

    type="button"
    modalProps={{ title: "hi" }}
    handleSubmit={(fieldValues, hideModal, toggleModalVisible) => {
      console.log(fieldValues);

    }}
    renderModalContent={form => {

    }}
  />
