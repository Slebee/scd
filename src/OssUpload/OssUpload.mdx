# OssUpload

> 4.0 上传文件特定组件,基于 antd Upload 重写其上传方法

## Basic Usage

注意，目前封装后只需要传入 data 对象，需要填写的 data 数据如下

```
{
  businessId (integer, optional): 业务Id,会附加到上传的文件名中,方便标识 ,
  businessType (string, optional): 业务类型,会附加到上传的文件名中,方便标识 ,
  companyId (integer): 公司Id ,
  ext (string, optional): 预留扩展字段 ,
  moduleCode (string): 模块码 ,
  productLineId (integer): 产品线Id ,
  userId (integer, optional): 对应的用户Id
}
```
