# StandardForm

> 提供一个通过(按钮|文字|自定义元素)触发 Modal 的弹出

## Basic Usage

目前 fields 接收以下类型

- Input
- AsyncSelect
- SearchSelect
- RadioGroup
- TextArea
- OssUpload
- RangeMoneyInput
- ProductLineSelect
- DatePicker
- Cascader
- CompanySearchSelect
- Select
- RangePicker

```javascript
// fields 元素
{
  id: string, // form.getFieldDecorator 的第一个参数id值
  span: number, // Col标签占用的span值,默认值8
  type: string, // 输入框类型，具体参照上面 如: Input
  formItemOptions: object, // formItem的 props 如: label
  decoratorOptions: object, // form的decorator的options 如: rules: []
  componentOptions: object, // 真实组件的 props，如: placeholder 等
}
```
